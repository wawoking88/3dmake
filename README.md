# Scratch風 3Dゲームメーカー 完全作り直し版

## 方針

以前の版では、3D空間のつまみ当たり判定とカメラ操作がぶつかっていました。
今回は完全に1から作り直し、つまみを3D空間ではなく、ステージ上のSVGオーバーレイで表示しています。

これにより、
- つまみを押したらカメラが動く
- つまみを掴んだまま解除されない
- タッチが反応しなくなる
- 公式TransformControlsの内部Raycaster問題

を避けています。

## 入っている機能

- スプライト追加ウィンドウ
  - 直方体
  - 球
  - 円柱
  - Capsule
  - 円すい
- 最初からCameraスプライトあり
- 実行中はCameraスプライトの位置・向きが実際の画面になる
- ステージ上のSVGつまみ
  - 移動：矢印
  - スケール：四角つまみ
  - 向き：リング
- タッチ対応
- 右側プロパティで位置/向き/大きさXYZ/色を編集
- Blockly対応
- if文と「キーが押された」を分離
- ↑↓←→のキー表示対応
- 停止すると、実行開始時の全スプライト情報に戻る

## 起動方法

`index.html` をブラウザで開いてください。

※ Three.jsとBlocklyをCDNから読み込むので、インターネット接続が必要です。
