# Scratch風 3Dゲームメーカー Blockly版

## 今回追加したもの

- スプライト追加は「＋」ボタンからウィンドウで選択
  - 直方体
  - 球
  - 円柱
  - Capsule
  - 円すい
- 最初から Camera スプライトを追加
- 実行中は Camera スプライトの位置・向きが実際のカメラになる
- カメラの向きが白い線と視野フレームで分かる
- ステージ上で選択スプライトを直接操作
  - 移動
  - 向き変更
  - スケール
- Blockly に変更
- if文と「キーが押された」を分離
- キー選択は ↑ ↓ ← → の表示に対応

## 起動方法

`index.html` をブラウザで開いてください。

## 注意

Three.js と Blockly を CDN から読み込んでいます。
インターネット接続が必要です。

## Blocklyの商用利用について

Blockly は公式ドキュメントで、アプリにブロック式コードエディタを追加できるWebライブラリとして説明されています。
コードサンプルは Apache 2.0 ライセンスです。
商用利用する場合は、Blockly本体や関連ライブラリのライセンス表記を残す前提で使うのが安全です。
