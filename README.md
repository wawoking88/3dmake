# Scratch風 3Dゲームメーカー Blockly修正版

## 修正内容

- エラーが出たとき、画面上に実際のエラー文を表示
- Blocklyの読み込みを1本化して安定化
- TransformControlsを使わず、独自の簡易3D操作ガイドに変更
  - 赤線：X
  - 緑線：Y
  - 青線：Z
- 移動 / 向き変更 / スケール変更に対応
- ＋ボタンから、直方体・球・円柱・Capsule・円すいを追加
- 最初からCameraを追加
- 実行中はCameraの位置と向きが実際の画面になる
- Blocklyで、if文と「キーが押された」を分離
- キー表示は ↑ ↓ ← → に対応

## 起動方法

`index.html` を開いてください。

※ Three.jsとBlocklyをCDNから読むため、インターネット接続が必要です。
